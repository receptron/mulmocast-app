# MulmoCast App v1.0.10 Release Notes

## 新機能

### Voice Clone (音声クローン)
ElevenLabs Voice Clone機能を新規追加しました。自分の声やカスタム音声をクローンしてナレーションに使用できます。

- ドラッグ&ドロップまたはクリックでの音声ファイルアップロード (最大10MB)
- クローンボイスの作成・削除・名前変更
- プレビュー再生機能
- スピーカー選択でスタンダードボイスと一緒に選択可能
- 音声クローン上限エラー時のわかりやすいエラーメッセージと管理画面へのリンク

### TTS (Text-to-Speech) 拡張

#### ElevenLabs
- **音声オプション追加**: speed(速度)、stability(安定性)、similarity_boost(類似度)のパラメータを調整可能
- グローバル設定とビート単位での上書きに対応
- 各パラメータにツールチップで説明を表示

#### Gemini TTS
- **instructionフィールド対応**: 読み上げスタイルを指定可能
- ビート単位での音声オプション上書きに対応

### チャット設定の保存
- チャット会話モードとスタイルテンプレート選択がセッションを跨いで保持されるように
- プロジェクト単位でのチャット設定保存に対応
- グローバル設定をデフォルトとして使用するハイブリッド方式

### UI/UX改善
- LLMによるスクリプト生成時にタイトルも自動生成
- スタイルタブにスピーチ/画像/動画/BGM/サイズのサマリー表示を追加 (Pro)
- 設定画面のAPIキー説明に対応機能を明記 (BGM生成、音声クローン、TTS)

## バグ修正
- BGM管理ページから離脱時にBGM再生が停止しない問題を修正
- Voice Clone操作後にUIが更新されない問題を修正
