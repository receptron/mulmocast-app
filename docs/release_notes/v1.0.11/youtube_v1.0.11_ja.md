MulmoCast v1.0.11 をリリースしました。

MulmoCastは、シンプルなスクリプトからAIで画像・音声・動画を自動生成し、マルチメディアプレゼンテーションを作成するデスクトップアプリです。

主な更新内容：
Azure OpenAI Service
- Azure経由でOpenAIの画像生成・音声合成・翻訳機能を利用可能。サービスごとにAPIキーとベースURLを個別設定。

Vertex AI 対応
- Google Cloud Vertex AI経由の画像生成（Imagen）・動画生成（Veo）に対応。デフォルトのProject ID・Locationを設定可能。

字幕分割（Caption Split）
- 句読点や記号で字幕を自動分割。スタイル設定でON/OFF切り替え。

Nijivoice TTS 削除
- サービス終了（2026年2月4日）に伴い削除。利用可能なTTS：OpenAI、ElevenLabs、Gemini、Google、Kotodama。
