## Description

```
MulmoCast v1.0.14 をリリースしました。

MulmoCastは、シンプルなスクリプトからAIで画像・音声・動画を自動生成し、マルチメディアプレゼンテーションを作成するデスクトップアプリです。

主な更新内容：
新しいAIモデル
- 19種類のAIモデルを新たに追加。画像生成モデル13種類（gpt-image-2、gpt-image-1-mini、Flux 2、Ideogram V3、SD 3.5 Large、Luma Photonなど）と、動画生成モデル6種類（Seedance 2.0、Hailuo 2.3、PixVerse v5など）。

並列実行数の設定UI
- 画像・動画・音声の同時生成数を、設定画面のデフォルト値とプロジェクト単位の両方で指定可能に。APIプランのレート制限に合わせて調整できます。

利用可能モデルの更新
- dall-e-3 と Imagen 4.0 が選択肢から外れました。オンボーディングのデフォルトも gpt-image-1-mini に変更しています。
```
