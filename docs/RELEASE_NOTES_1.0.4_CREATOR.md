# MulmoCast 1.0.4 リリースノート（クリエーター向け）

リリース日: 2025年11月12日

## 🎉 新機能

### メディア管理の改善
- **メディアモーダルのダウンロード機能を追加** - 生成された画像や音声をより簡単にダウンロードできるようになりました

### LLMモデル設定の改善
- **モデル選択の柔軟性が向上** - チャット機能で使用するLLMモデルをより細かく設定できるようになりました

## 🔧 改善点

### 安定性の向上
- **起動時のクラッシュを修正** - アプリケーション起動時の安定性が大幅に向上しました
- **エラーメッセージの改善** - エラーが発生した際に、より分かりやすいメッセージが表示されるようになりました

### ユーザーインターフェース
- **設定画面の改善** - より使いやすく、分かりやすい設定画面になりました
- **メニューの更新** - アプリケーションメニューが整理され、必要な機能により素早くアクセスできるようになりました

### パフォーマンス
- **MulmoCast エンジンを 2.0.0 に更新** - コンテンツ生成の品質とパフォーマンスが向上しました

## 🐛 バグ修正

- 起動時にクラッシュする問題を修正
- 設定の読み込みと保存に関する問題を修正
- 型の不整合による予期しないエラーを修正
- 国際化（日本語/英語）の表示に関する問題を修正

## 📝 その他の変更

- 依存パッケージを最新バージョンに更新し、セキュリティと安定性を向上
- アプリケーション全体の型安全性を強化

---

## アップデート方法

MulmoCast は自動更新に対応しています。アプリケーションを起動すると、新しいバージョンが利用可能な場合は自動的にダウンロードされ、次回起動時に適用されます。
